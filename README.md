# IEEE BIBM 2026 Workshop Site

Static conference workshop website template for GitHub Pages.

## Files

- `index.html` - page structure and content
- `styles.css` - visual design and responsive layout
- `script.js` - subtle scroll reveal behavior
- `assets/hero.png` - bitmap hero image

## Customize

1. Replace the placeholder workshop title, organizer names, and contact details.
2. Add your real dates, submission links, and schedule.
3. Commit the folder to a GitHub repository and enable GitHub Pages from the root branch.

## Notes

- This site uses relative paths, so it works as a GitHub Pages project site.
- The contact and FAQ sections are safe placeholders until your final CFP details are ready.
