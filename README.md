# Explainable and Trustworthy AI for Clinical Decision Support Systems

Static conference workshop website for GitHub Pages.

## Files

- `index.html` - page structure and content
- `styles.css` - visual design and responsive layout
- `script.js` - subtle scroll reveal behavior
- `assets/hero.png` - bitmap hero image

## Customize

1. Replace any placeholder links with the final workshop page and submission portal.
2. Add the live program order, speaker list, and paper links when they are finalized.
3. Commit the folder to a GitHub repository and enable GitHub Pages from the root branch.

## Notes

- This site uses relative paths, so it works as a GitHub Pages project site.
- The site is already populated with the approved workshop title, dates, organizers, and technical committee.
